Learning Management System (Capstone Project)

Project Overview
This project is a scalable LMS built to help matric rewrite students. It features student progress tracking, an analytics engine, and an e-commerce module for course enrollment.

Tech Stack
-Languages: PHP, JavaScript, SQL, HTML/CSS

-Database: MySQL

-Tools: Developed using Notepad++ and XAMPP

Key Features
* Real-time performance dashboards.
* Database-driven architecture.
* Integrated sales reporting for enrollments.
